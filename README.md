# hansung_cafeteria

한성대학교 학생 식단을 볼 수 있는 어플. 

학교 웹 사이트의 학생 식단 정보를 모바일 환경에서 보기 불편하여 만든 어플이다. 
Android 앱이며 UI 개발은 Jetpack compose를 이용하였고 MVVM 패턴을 적용하여 개발했다. 

## 기능

- 학생 식단을 학교 홈페이지에서 스크래핑하여 사용자에게 보인다.
- 오늘 날짜에 맞게 탭을 자동 이동한다.
